Enums

📄️ Alignment

Used in GUIs to specify to which side something should be aligned to

📄️ BLF Property Type

This enum is used only in BLF definitions, when you have to specify where an attribute will be saved to

📄️ BLF

This enum is used in BLF to specify what exact type an object should be saved as

📄️ CommandRule

Determines how the Console should turn user input into parameters for Commands.

📄️ Filter

Describes a Texture's interpolation mode

📄️ GraphicsBackend

This enum is used in conjunction with the getBackend() function to determine the current graphics backend being used by the engine.

📄️ Key

The enum used in key related events

📄️ KeyboardAction

When certain keys are pressed, they send out additional events to GUIObjects. Certain key combinations also have their own corresponding KeyboardAction enums.

📄️ Log

Used with the function log, it tells the engine the severity of your log message. It currently is only used to pick the color used for that message.

📄️ TextureMode

Currently used exclusively for Tiles. It is used to tell the Tile shader how the texture should be treated
