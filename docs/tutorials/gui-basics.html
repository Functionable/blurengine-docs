GUI Basics

You will probably be making lots of GUIs in the course of making your game(s) with BlurEngine.

The GUI system right now is very simple and primitive, lacking comforts you may expect like anchoring elements to one corner/side or layout managers.

You are expected to do the work of positioning and sizing your GUIs yourself. If screen size is relevant to where your GUI is placed (i.e: it's on the top or right side of the screen) you may wish to link with the screenResize event.

On GUIs and GUIObjects​

All GUI elements inherit from GUIObject. Though, all GUIs inherit from GUIContainer which itself is a GUIObject.

GUIContainer contains methods to add and manage GUIObjects to it. Such as: GUIContainer:addObject(), GUIContainer:removeObject() and GUIContainer:getChildren()

Making the GUIs​

note

The distinction between GUI.new() and GUI.newObject() is a leftover from an earlier version of blur where GUI didn't inherit from GUIObject.

These functions may get merged into a single function in another release.

An instance of a GUI must be made using GUI.new(). And an instance of a GUIObject must be made using GUI.newObject().

Here is an example snippet of code demonstarting this, where we will be making a simple Button and Label:

mainGUI = GUI.new("GUI")
mainGUI.Position = Vector2(300,300)
mainGUI.Size = Vector2(200,200)

titleGUI = GUI.newObject("Label")
titleGUI.Position = Vector2(60,150)
titleGUI.Size = Vector2(100,50)
titleGUI.Text = "Cool button"

buttonGUI = GUI.newObject("Button")
buttonGUI.Position = Vector2(25,50)
buttonGUI.Size = Vector2(150,75)
buttonGUI.Text = "Click me!"

Adding objects to GUIs​

When running the example above, you may notice that only the GUI appears. That's because objects need to be added to the GUIs to be displayed.

This can be done using the GUI:addObject() method.

-- Adds the title GUIObject and the button GUIObject to the main GUI class object.
mainGUI:addObject(titleGUI)
mainGUI:addObject(buttonGUI)

There we go!

This is a basic GUI, you can't interact with it yet and it doesn't look too pretty. To see how to do other things like changing fonts or adding button events, check out the GUI class. In order to learn how to add events to your newly made objects click here
