GUIContainer

Parent class: GUIObject

A GUIContainer stores and manages other GUIs and makes sure that events sent to the container also propagate to them.

Properties​

Vector2 Size Inherited from: GUIObject
The on-screen size of the object
bool Visible Inherited from: GUIObject
Determines whether the given object is drawn or not. Invisible objects cannot be interacted with.
bool Active Inherited from: GUIObject
When set to true on a given object, any interractions will not be checked (for example: mouse click, hover) and the object will be skipped. Which would for example result in objects being able to be interracted with through the one you set .Active to true on.
number Opacity Inherited from: GUIObject
Degree to which the objects's background is transparent
Color Color Inherited from: GUIObject
Default color of the object
Vector2 Position Inherited from: GUIObject
The on-screen position of the object relative to the parent container

Methods​

GUIContainer:pushFront( ) Inherited from: GUIObject
Pushes the object to the top (the object should now be displayed over every other object in the same container)
GUIContainer:pushBack( ) Inherited from: GUIObject
Pushes the object to the bottom (the object should now be displayed behind every other object in the same container)
GUIContainer:pushUp( ) Inherited from: GUIObject
Pushes the object nearer to the top (shifts its index within the container's order to be nearer to the top)
GUIContainer:pushDown( ) Inherited from: GUIObject
Pushes the object nearer to the bottom (shifts its index within the container's order to be nearer to the bottom)
GUIContainer:removeObject( GUIObject object )
Removes an object from the GUI
List[GUIObject] GUIContainer:getChildren( )
Returns a list of all objects present within the container
GUIContainer:addObject( GUIObject object )
Adds an object to the GUI

Events​

onEntered( ) Inherited from: GUIObject
Called when the mouse starts hovering over the object
onReleased( ) Inherited from: GUIObject
This is called when the mouse stops clicking the object
onMouseMove( Vector2 mouseMove ) Inherited from: GUIObject
Called when the mouse moves
onScrolled( number scrollX, number scrollY ) Inherited from: GUIObject
Called when using scrolling (either through the mouse wheel or a touchpad) while having the mouse be within the object
onPressed( ) Inherited from: GUIObject
This is called when the mouse clicks the object
onLeave( ) Inherited from: GUIObject
Called when the mouse stops hovering over the object
